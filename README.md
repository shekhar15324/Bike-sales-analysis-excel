# 🚴 Bike Sales Analysis Dashboard (Excel Project)

## 📊 Project Overview

This project analyzes customer demographic and lifestyle data to identify key factors influencing bike purchase behavior. Using Microsoft Excel, an interactive dashboard was created to uncover trends, customer segments, and actionable business insights.

The analysis helps businesses understand **which customers are more likely to purchase bikes** and supports data-driven marketing decisions.

---

## 🎯 Business Objective

* Analyze customer purchase patterns
* Identify high-value customer segments
* Understand factors affecting bike sales
* Provide insights for targeted marketing strategies

---

## 🛠 Tools & Techniques Used

* Microsoft Excel
* Data Cleaning & Transformation
* Pivot Tables & Pivot Charts
* Dashboard Design
* Data Visualization

---

## 📂 Dataset Information

The dataset contains customer-level information including:

* Age
* Gender
* Income
* Education Level
* Occupation
* Marital Status
* Number of Cars
* Number of Children
* Commute Distance
* Region
* Bike Purchase Status

Total Records Analyzed: **1000+ customers**

---

## 📈 Dashboard Features

* Regional Sales Analysis
* Income vs Purchase Comparison
* Age Group Segmentation
* Commute Distance Impact
* Customer Demographics Analysis
* Interactive filtering using Pivot Tables

---

## 🔍 Key Insights

* Customers with **2–5 mile commute distance** showed the highest purchase probability.
* The **Pacific region** recorded the strongest bike purchase rate.
* Customers owning **fewer cars** were significantly more likely to buy bikes.
* Individuals with **Bachelor’s and Graduate degrees** demonstrated higher adoption.
* Bike purchasing behavior was nearly **gender-neutral**.
* Middle-aged working professionals formed the primary customer segment.

---

## 💼 Business Recommendations

* Focus marketing campaigns on short-distance commuters.
* Target regions with higher conversion rates (Pacific region).
* Promote bikes as an alternative transport option for customers with fewer vehicles.
* Use demographic segmentation for personalized promotions.
